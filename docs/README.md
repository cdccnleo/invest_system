## 文档规范

1. **命名规范**：中英文混合，中文为主，使用 `_` 分隔
2. **版本管理**：文档文件纳入 Git 版本控制
3. **文档状态**：
   - ✅ 定稿 - 已完成并通过复核
   - 📝 待更新 - 需要补充或修订
   - 🔄 审核中 - 正在复核
4. **引用规范**：跨文档引用使用相对路径，外部资源使用完整路径
