## PIT #28: DeepSeek JSON 模式与 OpenAI 略异 (新)

**根因**:
1. OpenAI 支持 `response_format={"type": "json_object"}` 强制 JSON
2. DeepSeek **不支持该参数** (会忽略或报错)
3. DeepSeek 偶尔返回 markdown ```` ```json ... ``` ```` 块 (LLM 思考时常见)
4. 必须靠 prompt 强制 + 容错解析

**修复**:
```python
# 1. prompt 强制
system_prompt = """你是 Hermes Agent 投资分析助手...
返回严格 JSON, 不要任何其他文字:
{
  "affected_codes": ["code"],
  "direction": "positive|negative|neutral",
  "reasoning": "..."
}
"""

# 2. 容错解析 (复用 AInvest _parse_llm_response 逻辑)
def _parse_llm_json(content: str) -> Optional[Dict]:
    # 1. 直接 parse
    try: return json.loads(content)
    except: pass
    # 2. 提取 markdown ```json ... ``` 块
    import re
    m = re.search(r"```(?:json)?\s*(\{.*?\})\s*```", content, re.DOTALL)
    if m:
        try: return json.loads(m.group(1))
        except: pass
    # 3. 找第一个 {...} 块
    first = content.find("{")
    last = content.rfind("}")
    if first != -1 and last != -1:
        try: return json.loads(content[first:last+1])
        except: pass
    return None
```

**实测**: 4/4 case 通过 (含 markdown 块/garbage)

**PIT 教训**: 不同 LLM 的 JSON 模式 API 不一样, 永远要有 3 级解析: 直接→markdown→首个块。

## PIT #29: SemanticCache 启动耗 1-2s, 实战中用 LRU 即可 (新)

**根因**: AInvest `SemanticCache.__init__` 启动时尝试连接 pgvector (探测), 失败才降级到 LRU。这个探测耗时 1-2s。

**修复**:
```python
# 全局缓存客户端, 避免反复 init
_ainvest_client = None

def get_cached_ainvest_client():
    """PIT #29: 缓存客户端, 避免每次 import 1-2s 开销"""
    global _ainvest_client
    if _ainvest_client is None:
        _ainvest_client = get_ainvest_llm_client()
    return _ainvest_client
```

**实测**: 第二次调 0.8s (缓存命中) vs 第一次 1.7s (init + LLM)

**PIT 教训**: 单例模式 + 懒加载, 是任何"重的客户端"标准做法 (DB 连接池/LLM 客户端/HTTP 客户端)。

## PIT #30: 限流后 DeepSeek 返 429, 立即降级 Ollama (新)

**根因**: DeepSeek 限流策略 60 次/分钟 (默认), 超限返 429。AInvest 内部已做重试 + 降级, 但 V24-B2.1 仍需自己的"再降级"防御。

**修复**:
```python
# hpc.call_llm_for_event_match 双层降级
def call_llm_for_event_match(event_topic, holdings, use_ainvest=True):
    # 1. 优先 AInvest (含缓存命中)
    if use_ainvest:
        try:
            r = call_llm_for_event_match_ainvest(event_topic, holdings)
            if r: return r  # AInvest 命中
        except: pass  # AInvest 失败, 继续降级

    # 2. 降级 V24-B2 OpenAI 路径
    try:
        from openai import OpenAI
        # ...
    except: pass

    # 3. 都失败 → 返 None → hpc 关键词 fallback
    return None
```

**PIT 教训**: 双层 LLM 链 (AInvest 已有 DeepSeek→Ollama), 我们再套一层 (AInvest→OpenAI→None), 形成 3 级防御。

```python
    "688008": "DDR5 内存接口芯片全球第一 (36.8% 市占)",
    # ...
}
```

**实测**: SpaceX 事件 0 标的 → 加 hint 后 2 标的 (信维+西部材料) 100% 准确。
