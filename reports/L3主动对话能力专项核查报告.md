```python
capability_score = 0
if behavior_records >= 10:   capability_score += 2  # 初步行为数据
if behavior_records >= 50:   capability_score += 1  # 充足行为数据
if stress_tests:              capability_score += 2  # 已有压力测试结果
# 0-5 分 → ["沉睡", "萌芽", "激活", "成熟", "进阶", "完全"]
```
