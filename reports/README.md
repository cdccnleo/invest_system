## 报告规范

1. **命名规范**：中文为主，英文补充，使用 `_` 分隔
2. **版本管理**：报告文件纳入 Git 版本控制
3. **引用规范**：跨文档引用使用相对路径
4. **更新频率**：重大变更后更新，复核完成后追加
